## Overview

A Chrome browser extension that provides tools for recovering and managing iCloud data directly from the browser. Built with a focus on simplicity and data privacy — all processing happens locally.

## Features

- Browser-native iCloud data access
- Local processing — no data leaves your machine
- Intuitive UI for data recovery workflows

## Tech Stack

- **Platform:** Chrome Extension (Manifest V3)
- **Frontend:** JavaScript, HTML, CSS
- **API:** iCloud web services
