## 개요

브라우저에서 직접 iCloud 데이터를 복구하고 관리하는 Chrome 확장 프로그램입니다. 모든 처리가 로컬에서 이루어져 데이터 프라이버시를 보장합니다.

## 기능

- 브라우저 기반 iCloud 데이터 접근
- 로컬 처리 — 데이터가 외부로 전송되지 않음
- 직관적인 데이터 복구 UI

## 기술 스택

- **플랫폼:** Chrome 확장 (Manifest V3)
- **프론트엔드:** JavaScript, HTML, CSS
- **API:** iCloud 웹 서비스
