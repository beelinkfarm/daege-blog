## Overview

An AI-powered tool that generates self-contained, single-purpose web tools from natural language descriptions. Each tool is output as a standalone HTML file — no server, no dependencies, no setup required.

## How It Works

1. Describe what you need in plain English
2. The AI generates a complete HTML tool with embedded CSS and JavaScript
3. Open the file in any browser — it just works

## Tech Stack

- **Backend:** Python, FastAPI
- **AI:** Claude API for code generation
- **Output:** Self-contained HTML/CSS/JS files

## Key Features

- Natural language to functional tool in seconds
- Zero-dependency output files
- Built-in floating navigation buttons
- Responsive design out of the box
